OAuth App 설정

GitHub OAuth 로그인을 사용하려면 OAuth App을 등록해야 합니다.

  1. GitHub Developer Settings로 이동
  2. "New OAuth App" 클릭
  3. 다음 정보 입력:
    • Application name: Blog Editor
    • Homepage URL:
    • Authorization callback URL: (사용하지 않지만 입력 필요)
  4. ⚠️ Device Flow를 사용하므로 Client Secret은 필요 없습니다.
  5. "Register application" 클릭
  6. 생성된 Client ID를 아래에 입력
Client ID는 저장되어 다음에도 사용됩니다.
