# ⚠️ 로컬 파일 수정 안내

## 웹 에디터 사용을 권장합니다

이 블로그는 **웹 에디터**를 통해 포스트를 작성하도록 설계되었습니다.

### 🎯 웹 에디터 사용하기

1. **https://lebronisbest.github.io/editor/** 접속
2. GitHub OAuth 로그인 또는 Personal Access Token 입력
3. Quill 에디터로 시각적으로 포스트 작성
4. 저장 또는 발행

### ❌ 로컬 파일 직접 수정 시 주의사항

로컬에서 `_posts` 폴더의 파일을 직접 수정하면:

- 웹 에디터에서 저장할 때 충돌이 발생할 수 있습니다
- 변경사항이 덮어씌워질 수 있습니다
- 충돌 해결이 복잡해질 수 있습니다

### 💡 권장 사항

- **웹 에디터 사용**: 가장 안전하고 편리한 방법
- 로컬 수정이 필요한 경우:
  1. 웹 에디터에서 최신 버전 확인
  2. 로컬에서 수정 후 `git push`
  3. 웹 에디터에서 저장 시 충돌 경고 확인
