# 웹에서 포스트 편집하기

## 방법 1: GitHub 웹 에디터 사용 (간단)

### 새 포스트 작성하기

1. GitHub 저장소 페이지로 이동: `https://github.com/lebronisbest/lebronisbest.github.io`
2. `_posts` 폴더 클릭
3. "Add file" → "Create new file" 클릭
4. 파일명 입력: `YYYY-MM-DD-제목.md` (예: `2024-01-20-my-post.md`)
5. 아래 내용을 복사하여 붙여넣기:

```markdown
---
layout: post
title: "포스트 제목"
date: 2024-01-20
categories: [카테고리]
author: lebronisbest
---

여기에 본문을 작성하세요.
```

6. 하단의 "Commit new file" 클릭
7. 몇 분 후 자동으로 블로그에 반영됩니다!

### 기존 포스트 수정하기

1. GitHub 저장소에서 수정할 파일 클릭
2. 연필 아이콘(✏️) 클릭하여 편집 모드 진입
3. 내용 수정
4. "Commit changes" 클릭

## 방법 2: CMS 추가 (더 편리한 방법)

웹에서 더 편리하게 편집하려면 CMS(Content Management System)를 추가할 수 있습니다.

### 추천 CMS 옵션:

1. **Forestry.io** (무료, 사용 쉬움)
2. **Netlify CMS** (무료, 오픈소스)
3. **Decap CMS** (무료, Netlify CMS의 후속)

이 중 하나를 추가하면 웹 브라우저에서 블로그 관리 인터페이스를 사용할 수 있습니다.

---

**참고**: GitHub Pages는 자동으로 빌드되므로, 파일을 커밋하면 몇 분 후 자동으로 블로그에 반영됩니다.
