# 📝 마크다운 초간단 가이드

마크다운은 **정말 쉽습니다!** 5분이면 배울 수 있어요.

## 🎯 가장 많이 쓰는 것만 외우면 됩니다

### 1. 제목
```
# 큰 제목
## 중간 제목
### 작은 제목
```

### 2. 굵게 / 기울임
```
**굵게** 하려면 별표 2개
*기울임* 하려면 별표 1개
```

### 3. 리스트
```
- 항목 1
- 항목 2
- 항목 3
```

### 4. 링크
```
[링크 텍스트](https://example.com)
```

### 5. 줄바꿈
그냥 엔터 두 번 치면 됩니다!

---

## 💡 실제 예시

```markdown
오늘은 좋은 날이에요!

## 오늘 한 일

- 블로그 만들기
- 마크다운 배우기
- 커피 마시기

**정말 재미있네요!**
```

이게 전부입니다! 🎉

## 🚀 더 쉬운 방법

### 방법 1: 템플릿 복사하기
기존 포스트(`2025-11-18-welcome.md`)를 복사해서 내용만 바꾸세요!

### 방법 2: 일반 텍스트로 작성
마크다운 문법 몰라도 됩니다! 그냥 일반 텍스트로 작성해도 잘 보입니다.

### 방법 3: 온라인 에디터 사용
- **StackEdit**: https://stackedit.io/ (무료, 실시간 미리보기)
- **Dillinger**: https://dillinger.io/ (무료, 사용 쉬움)

이 에디터들에서 작성하면 자동으로 마크다운이 생성됩니다!
