## 🎯 가장 많이 쓰는 것만 외우면 됩니다

### 1. 제목
```
# 큰 제목
## 중간 제목
### 작은 제목
```

### 2. 굵게 / 기울임
```
**굵게** 하려면 별표 2개
*기울임* 하려면 별표 1개
```

### 3. 리스트
```
- 항목 1
- 항목 2
- 항목 3
```

### 4. 링크
```
[링크 텍스트](https://example.com)
```

### 5. 줄바꿈
그냥 엔터 두 번 치면 됩니다!
