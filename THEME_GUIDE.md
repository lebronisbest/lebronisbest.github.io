# 🎨 Beautiful Jekyll 테마 사용 가이드

## ✅ 적용 완료!

이제 **Beautiful Jekyll** 테마가 적용되었습니다. 이 테마는:
- ✨ 깔끔하고 모던한 디자인
- 📱 완전 반응형 (모바일 최적화)
- 🎯 사용하기 매우 쉬움
- 🔧 커스터마이징 가능

## 📝 포스트 작성하기

### 간단한 방법

1. **GitHub에서 직접 작성**
   - 저장소의 `_posts` 폴더로 이동
   - "Add file" → "Create new file"
   - 파일명: `2024-01-20-제목.md`

2. **템플릿 복사**
```markdown
---
layout: post
title: "포스트 제목"
subtitle: "부제목 (선택사항)"
date: 2024-01-20
author: lebronisbest
tags: [태그1, 태그2]
---

여기에 본문을 작성하세요.
```

3. **저장** → 자동으로 블로그에 반영!

## 🎨 테마 커스터마이징

`_config.yaml` 파일에서 다음을 수정할 수 있습니다:

- `title`: 블로그 제목
- `description`: 블로그 설명
- `avatar`: 프로필 이미지 URL
- `theme-color`: 테마 색상
- `social-network-links`: 소셜 링크

- **태그 사용**: 포스트에 태그를 추가하면 카테고리별로 정리됩니다

- **코드 블록**: Markdown의 코드 블록을 사용하면 자동으로 하이라이팅됩니다
