# 🚀 빠른 시작 가이드 (마크다운 몰라도 OK!)

## 가장 쉬운 방법

### 1단계: 템플릿 복사
`_posts/TEMPLATE.md` 파일을 복사하세요!

### 2단계: 파일명 변경
`2025-11-18-내-포스트.md` 형식으로 이름 변경

### 3단계: 내용만 수정
- 제목만 바꾸고
- 본문만 적으면 끝!

### 4단계: 푸시
```bash
git add _posts/파일명.md
git commit -m "새 포스트"
git push
```

## 💡 마크다운 몰라도 되는 이유

**그냥 일반 텍스트로 적어도 블로그에 잘 보입니다!**

예시:
```
오늘은 정말 좋은 날이에요.
블로그를 만들었어요.
정말 신나요!
```

이렇게만 적어도 충분합니다!
