# 🚀 빠른 시작 가이드

## 웹 에디터 사용하기 (가장 쉬운 방법!)

### 1단계: 웹 에디터 접속
**https://lebronisbest.github.io/editor/** 로 이동

### 2단계: GitHub 인증
- GitHub OAuth 로그인 (추천)
- 또는 Personal Access Token 입력

### 3단계: 포스트 작성
- 제목, 부제목, 날짜, 태그 입력
- Quill 에디터로 내용 작성 (네이버 블로그처럼!)
- 마크다운 문법 몰라도 OK!

### 4단계: 저장 또는 발행
- **저장**: 초안으로 저장
- **발행**: 블로그에 바로 반영

## ✨ 웹 에디터의 장점

- ✅ 마크다운 문법 몰라도 OK
- ✅ 시각적으로 작성 (WYSIWYG)
- ✅ 이미지 드래그 앤 드롭
- ✅ 실시간 미리보기
- ✅ 충돌 방지 기능

## ⚠️ 로컬 파일 수정은 권장하지 않습니다

로컬에서 직접 수정하면 웹 에디터와 충돌이 발생할 수 있습니다.
