# 🎯 네이버 블로그처럼 쉽게 사용하기

## 가장 쉬운 방법: 웹 관리 페이지 사용

### 1️⃣ 관리 페이지 접속
블로그에서 `/admin` 경로로 접속하거나, 아래 링크를 클릭하세요:
- **관리 페이지**: `https://lebronisbest.github.io/admin`

### 2️⃣ 새 포스트 작성
1. 관리 페이지에서 **"📝 새 포스트 작성하기"** 버튼 클릭
2. GitHub 편집 페이지가 열립니다
3. 파일명 입력: `2024-01-20-제목.md` (날짜-제목 형식)
4. 템플릿 복사해서 붙여넣기
5. 내용 작성
6. **"Commit new file"** 클릭
7. 완료! 몇 분 후 블로그에 자동 반영됩니다 ✨

### 3️⃣ 포스트 수정
1. 관리 페이지에서 **"📋 포스트 목록 보기"** 클릭
2. 수정할 포스트 클릭
3. 연필 아이콘(✏️) 클릭
4. 내용 수정
5. **"Commit changes"** 클릭

## 📝 포스트 작성 템플릿

아래 템플릿을 복사해서 사용하세요:

```markdown
---
layout: post
title: "포스트 제목"
date: 2024-01-20
categories: [일반]
author: lebronisbest
---

여기에 본문을 작성하세요.

## 소제목

- 리스트 항목 1
- 리스트 항목 2

**굵게**, *기울임*, `코드` 사용 가능!

### 이미지 추가
![이미지 설명](이미지주소)

### 링크 추가
[링크 텍스트](https://example.com)
```

## 🎨 간단한 문법

| 기능 | 문법 | 결과 |
|------|------|------|
| 굵게 | `**텍스트**` | **텍스트** |
| 기울임 | `*텍스트*` | *텍스트* |
| 제목 | `## 제목` | 큰 제목 |
| 리스트 | `- 항목` | • 항목 |
| 링크 | `[텍스트](주소)` | [텍스트](주소) |

## 💡 팁

1. **날짜는 자동**: 오늘 날짜를 자동으로 입력하려면 `date: 2024-01-20` 부분만 수정
2. **카테고리**: `[일반]`, `[개발]`, `[일상]` 등 자유롭게 설정
3. **이미지**: 이미지는 GitHub에 업로드하거나 외부 링크 사용
4. **자동 저장**: 커밋하면 자동으로 블로그에 반영 (1-5분 소요)
