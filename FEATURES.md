# 블로그 기능 가이드

애플 스타일의 미니멀하면서도 강력한 기능을 갖춘 Jekyll 블로그입니다.

## 주요 기능

### 1. 다크 모드 지원
- **자동 테마 전환**: 우측 상단의 토글 버튼으로 라이트/다크 모드 전환
- **설정 저장**: 브라우저에 테마 설정이 자동 저장됨
- **부드러운 전환**: 모든 색상이 CSS 변수로 관리되어 매끄러운 전환 효과

### 2. 검색 기능
- **실시간 검색**: 포스트 목록 페이지에서 제목, 부제목, 본문, 태그 검색 가능
- **즉시 반영**: 입력과 동시에 결과가 필터링됨
- **애니메이션**: 검색 결과가 부드럽게 나타남

### 3. 애니메이션 효과
- **페이드인 애니메이션**: 페이지 로드 시 요소들이 순차적으로 나타남
- **호버 효과**: 포스트 카드, 버튼, 태그에 마우스를 올리면 반응
- **스크롤 애니메이션**: 스크롤 시 요소들이 자연스럽게 나타남
- **부드러운 스크롤**: 앵커 링크 클릭 시 스무스 스크롤

### 4. 반응형 디자인
- **모바일 최적화**: 모든 화면 크기에서 완벽하게 작동
- **터치 친화적**: 모바일 기기에서 최적의 경험
- **유연한 레이아웃**: 다양한 기기에 맞춰 자동 조정

### 5. 사용자 경험 개선
- **Back to Top 버튼**: 스크롤 시 우측 하단에 나타나는 상단 이동 버튼
- **이미지 레이지 로딩**: 이미지가 부드럽게 로드됨
- **커스텀 404 페이지**: 페이지를 찾을 수 없을 때 친절한 안내

### 6. 애플 스타일 디자인
- **미니멀한 인터페이스**: 불필요한 요소를 제거한 깔끔한 디자인
- **SF 폰트 스택**: 애플 시스템 폰트 우선 사용
- **세련된 타이포그래피**: 가독성 높은 폰트 크기와 줄 간격
- **미세한 애니메이션**: 모든 인터랙션에 부드러운 애니메이션

## 색상 시스템

### 라이트 모드
- 배경: 화이트 (#ffffff)
- 텍스트: 다크 그레이 (#1d1d1f)
- 강조 색상: 블루 (#0071e3)

### 다크 모드
- 배경: 블랙 (#000000)
- 텍스트: 라이트 그레이 (#f5f5f7)
- 강조 색상: 라이트 블루 (#2997ff)

## 사용 방법

### 새 포스트 작성
1. `_posts` 폴더에 `YYYY-MM-DD-제목.md` 형식으로 파일 생성
2. Front Matter 작성:
   ```yaml
   ---
   layout: post
   title: "포스트 제목"
   subtitle: "부제목 (선택사항)"
   date: YYYY-MM-DD
   tags: [태그1, 태그2]
   ---
   ```
3. 마크다운으로 내용 작성

### 이미지 추가
- `assets/img/` 폴더에 이미지 업로드
- 포스트에서 `![설명](/assets/img/파일명.jpg)` 형식으로 삽입

### 다크 모드 토글
- 우측 상단의 원형 버튼 클릭
- 설정은 자동으로 저장됨

### 포스트 검색
- 포스트 목록 페이지에서 검색창 사용
- 제목, 내용, 태그로 검색 가능

## 성능 최적화

- **CSS 변수 사용**: 테마 전환 시 빠른 성능
- **Defer 스크립트 로딩**: JavaScript가 페이지 로딩을 방해하지 않음
- **이미지 최적화**: 레이지 로딩으로 초기 로딩 속도 개선
- **최소한의 의존성**: 외부 라이브러리 최소화

## 접근성

- **키보드 네비게이션**: 모든 기능을 키보드로 사용 가능
- **포커스 인디케이터**: 현재 포커스된 요소를 명확하게 표시
- **ARIA 레이블**: 스크린 리더를 위한 적절한 레이블 제공
- **색상 대비**: WCAG 기준을 만족하는 충분한 대비

## 브라우저 지원

- Chrome/Edge (최신 버전)
- Firefox (최신 버전)
- Safari (최신 버전)
- 모바일 브라우저 (iOS Safari, Chrome Mobile)

## 기술 스택

- **Jekyll**: 정적 사이트 생성기
- **Beautiful Jekyll**: 베이스 테마
- **CSS Variables**: 테마 관리
- **Vanilla JavaScript**: 외부 의존성 없는 순수 JavaScript
- **GitHub Pages**: 호스팅

## 커스터마이징

### 색상 변경
`assets/css/custom.css`의 `:root`와 `[data-theme="dark"]` 섹션에서 CSS 변수 수정

### 폰트 변경
`assets/css/custom.css`의 `body` 선택자에서 `font-family` 수정

### 애니메이션 조정
`assets/css/custom.css`의 `@keyframes` 섹션에서 애니메이션 수정

## 문제 해결

### 다크 모드가 작동하지 않음
- 브라우저 캐시 삭제
- JavaScript가 활성화되어 있는지 확인

### 검색이 작동하지 않음
- 포스트 목록 페이지(`/posts`)에서만 작동
- JavaScript가 활성화되어 있는지 확인

### 애니메이션이 너무 느림
- `assets/css/custom.css`에서 `transition` 시간 조정
- `prefers-reduced-motion` 미디어 쿼리 추가 고려
