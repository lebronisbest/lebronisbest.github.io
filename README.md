# myCPBL Dev Log

개발과 일상을 기록하는 블로그입니다.

## ✨ 주요 기능

- 🔍 **검색 기능**: 포스트 전체 검색
- 🏷️ **태그 시스템**: 태그별 포스트 분류 및 태그 클라우드
- 📅 **아카이브**: 날짜별 포스트 아카이브
- 🌙 **다크 모드**: 라이트/다크 모드 전환
- ⏱️ **읽기 시간**: 자동 계산된 읽기 시간 표시
- 📑 **목차**: 자동 생성되는 목차 (TOC)
- 💬 **댓글 시스템**: Utterances를 통한 GitHub 기반 댓글
- 🔗 **소셜 공유**: 트위터, 페이스북, 텔레그램 공유
- 📚 **관련 포스트**: 태그 기반 관련 포스트 추천
- ✍️ **웹 에디터**: GitHub OAuth 로그인을 통한 온라인 포스트 작성/편집

## 📝 포스트 작성하기

### 방법 1: 웹 에디터 사용 (추천)

1. https://lebronisbest.github.io/editor/ 접속
2. GitHub OAuth 로그인 또는 Personal Access Token 입력
3. 포스트 작성 및 발행

### 방법 2: GitHub에서 직접 작성

1. GitHub 저장소의 `_posts` 폴더로 이동
2. "Add file" → "Create new file" 클릭
3. 파일명: `2025-11-18-제목.md` 형식
4. `_posts/TEMPLATE.md` 파일을 참고하여 작성
5. "Commit new file" 클릭 → 자동으로 블로그에 반영!
