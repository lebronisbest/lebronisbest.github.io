## 📝 포스트 작성하기

### 가장 쉬운 방법

1. GitHub 저장소의 `_posts` 폴더로 이동
2. "Add file" → "Create new file" 클릭
3. 파일명: `2024-01-20-제목.md` 형식
4. 아래 템플릿 복사:

```markdown
---
layout: post
title: "포스트 제목"
subtitle: "부제목 (선택사항)"
date: 2024-01-20
author: lebronisbest
tags: [태그1, 태그2]
---

여기에 본문을 작성하세요.
```

5. "Commit new file" 클릭 → 자동으로 블로그에 반영!
