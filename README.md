# myCPBL Dev Log

개발과 일상을 기록하는 블로그입니다.

## ✨ 주요 기능

- 🔍 **검색 기능**: 포스트 전체 검색
- 🏷️ **태그 시스템**: 태그별 포스트 분류 및 태그 클라우드
- 📅 **아카이브**: 날짜별 포스트 아카이브
- 🌙 **다크 모드**: 라이트/다크 모드 전환
- ⏱️ **읽기 시간**: 자동 계산된 읽기 시간 표시
- 📑 **목차**: 자동 생성되는 목차 (TOC)
- 💬 **댓글 시스템**: Utterances를 통한 GitHub 기반 댓글
- 🔗 **소셜 공유**: 트위터, 페이스북, 텔레그램 공유
- 📚 **관련 포스트**: 태그 기반 관련 포스트 추천
- ✍️ **웹 에디터**: GitHub OAuth 로그인을 통한 온라인 포스트 작성/편집

## 📝 포스트 작성하기

**웹 에디터를 사용해주세요!** 🎉

1. https://lebronisbest.github.io/editor/ 접속
2. GitHub OAuth 로그인 또는 Personal Access Token 입력
3. Quill 에디터로 시각적으로 포스트 작성 (네이버 블로그처럼!)
4. 저장 또는 발행 버튼 클릭

### ⚠️ 중요 안내

- **로컬 파일 직접 수정은 권장하지 않습니다**
- 웹 에디터를 사용하면 충돌 없이 안전하게 작성할 수 있습니다
- 로컬에서 수정하면 웹에서 저장할 때 충돌이 발생할 수 있습니다
